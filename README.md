# 무조건 클리어 클론다이크 ♠♥♦♣

**언제 시작해도 반드시 끝까지 풀 수 있는** 클론다이크(Klondike) 솔리테어 카드 게임입니다.
설치 없이 웹 브라우저에서 바로 즐길 수 있고, 휴대폰 홈 화면에 앱처럼 추가할 수도 있어요.

> 보통 솔리테어는 무작위로 카드를 섞으면 약 20%는 절대 못 푸는 판이 나옵니다.
> 이 게임은 **컴퓨터가 미리 "풀 수 있는 판"인지 확인한 판만** 보여주기 때문에, 막혀서 못 깨는 일이 없어요.

---

## 🎮 게임이 뭔가요? (솔리테어 처음이라면)

카드 52장을 규칙에 맞게 정리해서 **4개의 칸(파운데이션)에 무늬별로 A→K 순서로 모두 쌓으면 승리**하는 1인용 카드 게임입니다.

화면은 크게 세 부분으로 나뉩니다.

- **위 왼쪽 (스톡 / 웨이스트)**: 뒤집어 둔 카드 더미. 탭하면 한 장씩 펼쳐집니다.
- **위 오른쪽 (파운데이션 4칸 ♠♥♦♣)**: 여기에 같은 무늬끼리 **A(에이스) → 2 → 3 → … → K(킹)** 순서로 쌓습니다. 여기에 모든 카드를 다 올리면 끝!
- **아래 (타블로 7줄)**: 메인 작업 공간. 카드를 옮겨가며 길을 만듭니다.

### 기본 규칙
- **타블로(아래 7줄)**: 카드를 **색이 번갈아(빨강↔검정) 가며 숫자가 1씩 작아지게** 올립니다.
  예) 검정 8 위에는 빨강 7을 올릴 수 있어요.
- **파운데이션(위 4칸)**: **같은 무늬**로 **A부터 K까지 순서대로** 쌓습니다.
- 빈 타블로 줄에는 **K(킹)**부터 시작할 수 있습니다.
- 뒤집혀 있던 카드는 그 위 카드를 치우면 자동으로 앞면이 보입니다.

---

## 🕹️ 어떻게 조작하나요?

### 휴대폰 / 태블릿 (터치)
- **카드 옮기기**: 카드를 손가락으로 **누른 채 끌어다** 원하는 자리에 놓습니다.
  (이어진 카드 묶음도 함께 끌려옵니다. 놓을 수 있는 자리는 노란 테두리로 표시돼요.)
- **파운데이션으로 한 번에 올리기**: 카드를 **빠르게 두 번 톡톡(더블탭)** 하면 위 칸으로 자동으로 올라갑니다.
- **카드 펼치기**: 왼쪽 위 더미를 **탭**하면 카드가 한 장씩 나옵니다. 더미가 비면 ↻ 를 눌러 다시 채웁니다.

### 컴퓨터 (마우스)
- **카드 옮기기**: 마우스로 카드를 끌어다 놓기(드래그 앤 드롭).
- **자동 올리기**: 카드를 **더블클릭**하면 파운데이션으로 자동 이동.
- **펼치기**: 왼쪽 위 더미를 클릭.

## 🔘 버튼 설명 (아주 쉽게)

화면 위쪽에 버튼들이 한 줄로 있고, 맨 오른쪽에는 작은 **램프(●)** 가 있습니다.

| 버튼 | 하는 일 |
|---|---|
| **NEW** | 새로운 판을 시작합니다. (항상 풀 수 있는 새 판) |
| **RESET** | 지금 판을 **처음 상태로 되돌려** 다시 시작합니다. (같은 카드 배치) |
| **UNDO** | 방금 한 행동을 **한 단계 되돌립니다.** |
| **HINT** | **"이렇게 두면 끝까지 풀 수 있어요"** 하는 다음 한 수를 알려줍니다. (카드와 목적지를 깜빡여 표시) |
| **SAVE** | 지금 판의 **'판 코드'를 복사**합니다. 친구에게 보내거나 보관했다가 똑같은 판을 다시 즐길 수 있어요. |
| **LOAD** | 복사해 둔 **판 코드를 붙여넣어** 그 판/상황을 그대로 불러옵니다. |
| **SNAP** | **지금 이 순간의 진행 상황**(카드 위치까지 전부) 코드를 복사합니다. 나중에 LOAD로 그대로 재현돼요. |
| **AUTO** | 남은 카드를 **자동으로 끝까지 정리**해 줍니다. 진행 중에는 화면을 탭하거나 STOP으로 멈출 수 있어요. |

### 램프(●)는 뭔가요?
맨 오른쪽 동그란 점은 **지금 상태에서 클리어가 가능한지**를 알려주는 신호등이에요.
- 🟢 **초록불**: 아직 끝까지 풀 수 있어요. (안심!)
- 🔴 **빨간불**: 지금 배치로는 더 이상 풀 수 없어요. → **UNDO로 되돌리거나 RESET** 하세요.

> 처음 판은 항상 풀 수 있게 만들어지지만, 플레이하다 막다른 수를 두면 빨간불이 될 수 있습니다. 그럴 땐 UNDO!

## 💡 힌트는 "이유"까지 알려줘요

**HINT** 버튼을 누르거나 잠시 가만히 있으면 뜨는 해설창은, 다음 한 수만 짚어주는 게 아니라 **왜 그 수가 좋은지**까지 설명합니다.

- **무엇을 / 어디로**: 옮길 카드와 목적지를 화면에서 깜빡여 보여줍니다.
- **받을 카드 안내**: 다리를 놓는 수라면 **그 위에 올라올 카드의 색과 숫자**를 알려줘요.
  예) *"이 카드를 옮기면 그 위에 **검은색 6** 을 받을 수 있어요."*
- **이득과 손해**: 파운데이션에 올리는 수는 **이득(초록 글씨)** 과 **손해(빨간 글씨)** 로 구분해 보여줍니다.
  - 이득: 파운데이션 한 칸 진행 · 숨은 카드 공개 · 빈 칸(King 자리) 만들기 등
  - 손해: 그 카드를 타블로의 '다리'로 못 쓰게 됨
- **"다음" 버튼**: 추천 수를 대신 둬 주고 곧바로 다음 수를 보여줍니다. 계속 누르면 **끝까지 따라가며 클리어**할 수 있어요.

### 🤫 공정한 힌트 — 숨은 카드는 알려주지 않아요

힌트는 **여러분이 실제로 볼 수 있는 정보만으로** 설명합니다. **뒤집혀 있는(엎어진) 카드의 정체는 절대 미리 알려주지 않아요.**

- "안전하게 올린다"는 안내는, 그 카드를 받을 반대색 카드가 **이미 파운데이션에 다 올라간 게 눈에 보일 때만** 합니다. 보이지 않으면 안전을 단정하지 않고 이득·손해만 알려줘요.
- 숨은 카드를 까는 수는 *"엎어진 카드가 드러나요"* 라고만 하고, 그게 무슨 카드인지는 말하지 않습니다.
- 단, **스톡 카드**는 한 바퀴 넘겨 보면 누구나 알 수 있으므로 예외로 이름을 알려줍니다.

## 💾 자동 저장 (이어서 하기)

- 한 수 둘 때마다 **진행 상황이 휴대폰/브라우저에 자동으로 저장**됩니다.
- 게임을 껐다가 다시 열면 **하던 판이 그대로 이어집니다.** 따로 저장 버튼을 누를 필요가 없어요.
- 한 판을 클리어하면 저장 기록은 자동으로 비워지고, 다음에 새 판으로 시작합니다.

---

## 📲 휴대폰에 앱처럼 설치하기 (선택)

설치 파일 없이도, 브라우저에서 "홈 화면에 추가"를 하면 앱 아이콘처럼 쓸 수 있어요.

- **안드로이드(크롬)**: 게임 접속 → 우측 위 **⋮ 메뉴 → "홈 화면에 추가"**
- **아이폰(사파리)**: 게임 접속 → 아래 **공유 버튼 → "홈 화면에 추가"**

설치하면 전체화면으로 실행되고, 한 번 연 뒤에는 **인터넷이 없어도 플레이**할 수 있습니다.

## ⚙️ 어떻게 "무조건 클리어"가 되나요? (궁금한 분만)

새 판을 만들 때, 화면 뒤에서 **솔버(푸는 프로그램)** 가 그 배치를 실제로 풀어 봅니다.
풀리는 판이면 보여주고, 안 풀리면 다른 배치로 다시 시도합니다.
무거운 계산은 **백그라운드(별도 작업 줄)** 에서 처리하기 때문에, 게임이 멈칫하지 않고 부드럽게 돌아갑니다.

- 게임 종류: 클론다이크, **한 장씩 뽑기(Draw 1)**
- HINT와 AUTO도 같은 솔버를 활용해 **항상 정답으로 이어지는 수**를 안내/실행합니다.
